# Typing Tester

An advanced typing speed test application built with HTML, CSS, and JavaScript.

## Features
- Multiple difficulty levels (Easy, Medium, Hard, Expert)
- WPM, Accuracy, Errors, and Level tracking
- Timer functionality
- Live result display
- Live character highlighting (green/red)
- More complex typing challenges


## Files
- `index.html` – main HTML file
- `style.css` – styles
- `script.js` – JavaScript logic
- `texts.json` – paragraph database
